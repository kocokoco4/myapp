## 1. プロジェクト概要

### 1.1 目的
個人作曲家が「雰囲気」から直感的に楽曲制作を始められるWebアプリ。
歌詞・コード・メロディを同一画面で管理し、DAW（LogicPro/GarageBand）に連携できる。

### 1.2 対象ユーザー
- ソロシンガーソングライター
- 趣味の作曲家
- DAW初心者（GarageBand/LogicProユーザー）
- 音楽理論をあまり知らない人

### 1.3 差別化ポイント
1. **雰囲気から作れる** — 感情×景色×テンション×関係性の掛け合わせで曲の骨格を自動生成
2. **歌詞×メロディが同画面** — 他アプリは歌詞と楽譜が別管理
3. **DAW連携** — MIDI/MusicXML出力でGarageBand/LogicProに持ち出せる
4. **AI伴奏** — コード進行からピアノ・ベース・ギター・ドラムの伴奏を自動生成

## 3. 非機能要件

### 3.1 パフォーマンス
- 初回ロード: 3秒以内（3G回線除く）
- ビルドサイズ: JS 700KB以下（gzip 210KB以下）
- Firestoreの読み書き: 1操作あたり500ms以内

### 3.2 セキュリティ
- Firebase Auth必須
- Firestoreルールでuidベースアクセス制御
- APIキーはサーバー側管理（Cloud Functions）
- XSS対策: React標準のエスケープ + dangerouslySetInnerHTMLは五線譜SVGのみ
- 決済情報: Stripe管理（当方で保持しない）

### 3.3 可用性
- Firebase Hosting: 99.95% SLA（Google Cloud）
- Firestore: 99.999% SLA
- オフライン: localStorageフォールバックで基本操作可能

### 3.4 法的要件
- 特定商取引法に基づく表記（/legal.html）※事業者情報要記入
- プライバシーポリシー（/privacy.html）
- 利用規約（/terms.html）
- 消費者契約法: 解約ボタンを設置
- 電気通信事業法: Cookie利用の通知をプライバシーポリシーに記載
